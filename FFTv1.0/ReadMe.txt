はじめに
　高速フーリエ変換用プログラムです．自己責任でご利用ください．

注意
  サンプリングデータの出力は次のように設定してください.
  ・音声数：ステレオ
  ・計測スケール: リニア
  ・インデックス（テキストファイル時のみ）: 無し
  ・ヘッダ情報を含める : 無し
  ・ヘッダー追記文:
  ・ステレオチャンネル記述欄: 左右を同じ行に
  メモリの動的確保失敗に関しての記述を単純に処理終了にしています．メモリを十分に確保したうえでご利用ください．

付属ファイル
・fft_define.h : FFT用マクロ定義＆構造体
・fft.c/.h : 高速フーリエ変換用関数群
・IO.c/.h : データ入力用関数群
・fft_test.c : fft実行ファイル（mainを含む）
・ReadMe.txt : このファイル
・sample-data3.txt : サンプリングデータ

コンパイル方法
> bcc32c .\fft_test.c .\fft.c .\IO.c
> gcc -o .\fft_test.exe .\fft_test.c .\fft.c .\IO.c
etc...

実行方法
> (実行ファイル) (サンプリングデータ) (サンプリング周波数)
> fft_test.exe sample-data3.txt 8000 > output.csv
